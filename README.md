# Sudoku_game
Sudoku is a popular logic-based puzzle played on a 9×9 grid, divided into nine smaller 3×3 boxes. The goal is to fill the grid with numbers 1 through 9 so that: - Each row contains all digits 1–9 without repetition. - Each column contains all digits 1–9 without repetition. - Each 3×3 box contains all digits 1–9 without repetition.
